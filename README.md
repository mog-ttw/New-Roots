# New Roots

A beautiful multi-page website for New Roots, an organization empowering refugees through digital skills training.

## Pages

- Home: Overview of New Roots mission and impact
- About Us: Story, mission, and values
- Our Program: Training pipeline and curriculum
- Partners: Corporate partners
- Success Stories: Graduate testimonials
- Contact: Get in touch!

## Tech Stack

- HTML5
- Tailwind CSS
- Vanilla JavaScript

## Getting Started

1. Clone the repository: `git clone git@github.com:mog-ttw/New-Roots.git`
2. Open `index.html` in your browser to view the site locally!
